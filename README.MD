Replaces the 2D fishing hook with a 3D one

xou can get it on [Modrinth](https://modrinth.com/resourcepack/fishing-hook-bobber-3d) or [Curseforge](https://www.curseforge.com/minecraft/texture-packs/fishing-hook-bobber-3d)

available with and without line at the top, in 3 different modes.

![overview of the differnt versions](https://raw.githubusercontent.com/fayer3/3d-bobber/refs/heads/master/images/overview.png)

compatible with Optifine/Sodium

not compatible with Iris/Optifine Shaders

has support for [Aquaculture 2](https://www.curseforge.com/minecraft/mc-mods/aquaculture)Â (the Aquaculture team has given me permission to use their textures, to add support in this pack)

![aquaculture support image](https://raw.githubusercontent.com/fayer3/3d-bobber/refs/heads/master/images/aquaculture.png)

has support for tinkers construct fishing  rods

to add a fishing bobbers of other mods, set the top left pixel of their texture to (255, 0, 255, 0) and the top right one to (0, 255, 255, 0)

there is a python scrip included to do this for you, to use it, run it with `python set.py <path to folder with texture>`
